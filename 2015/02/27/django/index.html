模板是自动转义的，关闭转义

Django模板为了防止XSS，缺省情况下会自动将HTML进行转义。

有的时候我们确实想输出一些含有HTML代码的内容（比如Web编辑器产生的文章内容数据），如何关闭自动转义？
方法1：使用

python

{ % autoescape off % }
